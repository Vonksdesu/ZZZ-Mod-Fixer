# ZZZ-Mod-Fixer by petrascyll

This is a project to update and fix the character mod hash IDs from the game Zenless Zone Zero (ZZZ) to the latest version of the game.

## Diclaimer

This project wasn’t entirely my own creation from the start. I simply continued working on it of my own volition, without being told to or pressured by anyone else.
